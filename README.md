# cheerio_weather

- 以cheerio實作氣象預報爬蟲

- Backend: NodeJS (API)

- Frontend: ReactJS (View)

## install

```
npm install
```

## run

```
npm start
```

## client

- http://localhost:3000/

## api

- http://localhost:3000/getWeather/1
- http://localhost:3000/getWeather/2
- http://localhost:3000/getWeather/3
